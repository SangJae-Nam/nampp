NAME
   NAM++ - typed-object-oriented language based on Racket

USAGE
   INSTALLING the source:
     Just download source from github(https://github.com/SangJae-Nam/nampp)

   MAKE executable file by source:
     1) Open "frame.rkt" by Racket
     2) Click "Racket"->"Make executable file"

   RUN NAM++ at Racket:
     1) Open "frame.rkt" by Racket
     2) Click "RUN"

   TEST FILES:
     Test files are located in "test" directory.
     So you can run and test these files.

SOFTWARE REQUIREMENTS
   Racket v5.0.0 or later.
